# CoinLab 一期功能范围

> 来源：产品功能脑图 · 2026-06-17  
> UI 主色：**黄 (#FFCC00) + 黑 (#000000)**

## 一期模块

| 模块 | 功能 |
|------|------|
| **基础** | 多语言、法币选项、帮助中心（服务协议/隐私政策）、客服 |
| **账户** | 登录/注册（第三方邮箱）、个人资料、安全设置（Google/邮箱/登录密码/支付密码）、KYC（Sumsub 或自审）、注销 |
| **合规** | 按 KYC/区号识别地区，展示模块或警告弹窗 |
| **Banner** | 固定轮播（活动/公告） |
| **行情** | K 线、自选、订单簿、市场深度、最新成交 |
| **现货交易** | 市价/限价单、订单管理（一期不对冲） |
| **钱包充提 & 资产** | BSC/TRC20 充提 USDT·BNB·TRX；资产余额；账单流水 |
| **运营后台** | 用户/现货/充提/合规管理（非 App 端） |

## 产品端

- **App：** iOS（TestFlight）、Android
- **Web：** 移动 H5（含 Telegram Mini App）、PC Desktop

## 后续阶段（非一期）

- **二期：** 服务端接口重构、Nx 账户资产迁移、新渠道站
- **三期：** 代理/经纪人、现货对冲、**合约**、站内信、Banner 动态配置
